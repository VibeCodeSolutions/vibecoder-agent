- Initial release of VibeCoder Agent
- AI-powered chat interface with local DeepSeek R1 model via Ollama

- Automatic error analysis and fix suggestions
- Terminal command execution
- Git operations (init, add, commit, branch)
- File operations (create, read, write)
- Dark theme chat UI
- Status bar indicator
- Conversation history management
